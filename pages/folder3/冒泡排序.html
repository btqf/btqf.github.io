什么是冒泡排序？ 所谓冒泡排序就是如同水中泡泡般，大的泡泡往下沉，小的泡泡往上升，即数组中相邻元素进行比较，前一个元素比后一个元素要大就进行交换。遍历一次后数组最后一个元素为最大值，之后循环遍历即可，

# 思路：

  1. 从第一个元素开始，让其与右边相邻元素比较，如果左边元素大于右边元素则交换并且左右下标同时向右移动，再次比较。
  2. 若左边元素小于右边元素，则右边元素向右移动。如此一次遍历，最右边元素为数组中最大
  3. 分别对剩下的元素依次进行第一步和第二步即可。

# 代码

function bubbleSort(arr) {
    const len = arr.length
    if (len < 2) return arr
    for (let i = 0; i < len; i++) {
        for (let j = 0; j < len - i - 1; j++) {
            if (arr[j] > arr[j+1]) {
                [ arr[j], arr[j+1] ] = [ arr[j+1], arr[j] ]
            }
        }
    }
    return arr
}

优化：假如从开始的第一对到结尾的最后一对，相邻的元素之间都没有发生交换的操作，这意味着右边的元素总是大于等于左边的元素，此时的数组已经是有序的了，我们无需再对剩余的元素重复比较下去了

function bubbleSort (arr) {
  const len = arr.length
  if (len < 2) return arr
  // 改进一：利用内外循环一加一减，简化越界问题
  for (let i = 0; i < len; i++) {
    // 改进二：利用标志，若flag没有发生改变即表示相邻元素没有发生调换，直接返回
    let flag = true
    for (let j = len - 1; j > i; j--) {
      if (arr[j-1] > arr[j]) {
        [ arr[j-1], arr[j] ] = [ arr[j], arr[j-1] ]
        flag = false
      }
    }
    if (flag) break
  }
  return arr
}
