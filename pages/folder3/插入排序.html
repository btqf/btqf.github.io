什么是插入排序？ 所谓插入排序就是类似于选择排序的改进，它不像选择排序一般每次遍历寻找无序集合中最小的元素，然后将其放入到有序集合之后。它是将遍历到的元素一个一个地插入到有序集合中恰当的位置。

# 思路：

  1. 将第一个元素设定为有序集合。
  2. 将无序集合的第一个元素设为inserted, 从后往前与有序集合中元素比较。所有比inserted大的元素后退一位，然后将其插入。
  3. inserted插入成功后，将待插入元素后退一位即可。

# 代码

function insertSort(arr) {
    const len = arr.length
    if (len < 2) return arr
    for (let i = 1; i < len; i++) {
        // 记录待插入元素
        let inserted = arr[i]
        let j = i - 1
        // 比待插入元素大的元素后退一位
        for (j; j >= 0 && arr[j] > inserted; j--) {
            arr[j+1] = arr[j]
        }
        arr[j+1] = inserted
    }
    return arr
}
